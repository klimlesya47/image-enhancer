# Улучшение изображений в браузере

Система улучшения изображений (яркость, контрастность, насыщенность) на основе
компактной нейросети.

Демо: https://github.com/klimlesya47/image-enhancer

## Как это работает

1. Пользователь загружает изображение (JPG, PNG, HEIC, BMP)
2. Изображение уменьшается до 224×224 и прогоняется через компактную CNN,
   которая предсказывает три параметра коррекции - brightness, contrast, saturation
3. Эти параметры применяются ко всему изображению в исходном разрешении
4. Вся обработка идёт в Web Worker - интерфейс не блокируется на всё время
   обработки, есть возможность отследить прогресс и прервать задачу

## Структура репозитория

```
src/            браузерное приложение (API, Web Worker, UI)
training/       генерация датасета и обучение ML-модели
samples/        эталонные изображения для тестирования
```

Бюджет "суммарный объём кода ≤ 10 МБ" считается только для собранной папки dist/
training/ и samples/ — вспомогательные материалы, не часть браузерного бандла, в этот бюджет не входят.

## Технологический стек

- TypeScript + Vite - браузерное приложение
- Web Worker - асинхронная обработка без блокировки интерфейса
- Ручной forward pass - вместо ONNX Runtime Web, чтобы уложиться в бюджет размера кода 
- PyTorch - обучение модели

## Запуск локально

```bash
npm install
npm run dev
```

Откроется `http://localhost:5173`.

## Сборка и проверка размера

```bash
npm run build
```

Проверить итоговый размер собранного приложения:

```bash
# Linux / macOS
du -sh dist/

# Windows PowerShell
Get-ChildItem -Recurse dist | Measure-Object -Property Length -Sum | ForEach-Object { "{0:N2} MB" -f ($_.Sum / 1MB) }
```

Текущий результат: 5.41 МБ.
